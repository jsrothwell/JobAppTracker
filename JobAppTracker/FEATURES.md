## ðŸŽ¯ Timeline View

The Timeline view provides a chronological history of your job application progress:

### What It Shows
- **Visual Timeline**: Each status change is displayed with:
  - Color-coded status indicator
  - Status icon (paperplane, people, star, etc.)
  - Exact date and time of change
  - Optional notes explaining the change
  - Days spent in current status

### How to Use
1. Select any job from the list
2. Click the "Timeline" tab in the detail panel
3. View the complete history of status changes
4. Most recent changes appear at the top

### Status Change Tracking
Every time you update a job's status, the app automatically:
- Records the date and time
- Saves any notes you add
- Creates a timeline entry
- Maintains full history

## ðŸ“‹ Kanban Board View

The Kanban Board provides an intuitive, visual way to manage your applications:

### Layout
Five columns representing each status:
- **Applied** (Blue)
- **Interviewing** (Orange)  
- **Offer** (Green)
- **Rejected** (Red)
- **Accepted** (Purple)

### Features
1. **Drag & Drop**
   - Grab any job card
   - Drag it to a different column
   - Status updates automatically
   - Timeline records the change

2. **Visual Cards**
   Each card displays:
   - Position title
   - Company name
   - Date applied
   - Location (if provided)
   - Salary range (if provided)

3. **Column Headers**
   - Status icon and name
   - Count of jobs in that status
   - Color-coded for quick identification

4. **Quick Actions**
   - Click any card to view full details
   - Drag to update status instantly
   - Add new jobs from the top bar

## ðŸ”„ View Switcher

Toggle between List and Kanban views:

### List View
- Best for: Searching, filtering, detailed browsing
- Features: Search bar, status filters, statistics, sortable list
- Layout: Split view with list on left, details on right

### Kanban View  
- Best for: Visual overview, quick status updates
- Features: Drag-and-drop, at-a-glance status, visual organization
- Layout: Full-screen columns

### Switching Views
Use the segmented control in the top bar:
- Click "List" for traditional list view
- Click "Kanban" for board view
- Your data is the same in both views

- See how long you've been in each status
- Review your application patterns
- Track response times from companies

   - Prepare better for discussions
